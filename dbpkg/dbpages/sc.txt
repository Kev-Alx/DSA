# Dashboard Component Showcase

Welcome to the **full feature showcase** for `DashboardPage`. This document demonstrates every supported Markdown element and custom component. Use the sidebar to jump between sections.

## Typography

Paragraphs render with comfortable `1.75` line-height and proper `1.25em` bottom margin, so long passages of text remain readable without feeling cramped.

You can use **bold text** for emphasis, _italic_ for tone, and even **_bold italic_** when you really need to stress a point. Inline `code snippets` render with a subtle pill background that keeps them legible inline.

Here is a [link to the Anthropic website](https://anthropic.com) demonstrating styled anchor rendering with a hover-opacity transition.

> Blockquotes are rendered with a left-border accent and muted italic text — great for callouts, citations, or highlighted notes from documentation.

## Tables

Tables are powered by `remark-gfm` and support left, centre, and right column alignment. Rows get a subtle hover highlight.

### Basic Table

| Component       | Type        | Default   | Description                               |
|-----------------|-------------|-----------|-------------------------------------------|
| `sidebar`       | `boolean`   | `true`    | Show the right-hand TOC sidebar           |
| `figureBorders` | `boolean`   | `false`   | Draw borders around images and charts     |
| `padding`       | `PaddingSize` | `"md"` | Controls outer content padding            |
| `defaultChartStyle` | `object` | `—`    | ECharts base config merged into all charts |
| `codeBlock`     | `object`    | `—`       | Code block theme and copy-button settings |

## Code Blocks

Code blocks use the theme from `settings.codeBlock.theme`. The header shows a language badge, macOS-style traffic-light dots, and an animated copy button (when `copyButton: true`).

### JavaScript

```javascript
import { DashboardPage } from "./Dashboard";

const content = `# Hello World`;

export default function App() {
  return (
    <DashboardPage
      md={content}
      settings={{
        codeBlock: { theme: "github-dark", copyButton: true },
        padding: "lg",
      }}
    />
  );
}
```

### TypeScript

```typescript
interface DashboardPageSettings {
  sidebar?: boolean;
  figureBorders?: boolean;
  padding?: "sm" | "md" | "lg" | "xl";
  defaultChartStyle?: Record<string, any>;
  codeBlock?: {
    theme?: "github-light" | "github-dark" | "dracula" | "monokai" | "solarized-light";
    copyButton?: boolean;
  };
}
```

### Shell

```bash
# Install required peer dependencies
npm install react-markdown remark-gfm rehype-raw \
            echarts echarts-for-react slugify unist-util-visit
```

### Python

```python
def deep_merge(target: dict, source: dict) -> dict:
    """Recursively merge source into target; source wins on conflicts."""
    out = {**target}
    for key, sv in source.items():
        tv = target.get(key)
        if isinstance(sv, dict) and isinstance(tv, dict):
            out[key] = deep_merge(tv, sv)
        else:
            out[key] = sv
    return out
```

### `DashboardPage` Props

| Prop       | Type                     | Description                      |
|------------|--------------------------|----------------------------------|
| `md`       | `string`                 | Markdown content to render       |
| `title`    | `string?`                | Page title (used in nav tab)     |
| `settings` | `DashboardPageSettings?` | Full settings object (see below) |

### `DashboardPageSettings`

```typescript
{
  sidebar?: boolean;          // default: true
  figureBorders?: boolean;    // default: false
  padding?: PaddingSize;      // "sm" | "md" | "lg" | "xl"  default: "md"
  defaultChartStyle?: object; // ECharts base options
  codeBlock?: {
    theme?: CodeTheme;        // 5 themes available
    copyButton?: boolean;     // default: true
  };
}
```
